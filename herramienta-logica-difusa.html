Lógica Difusa

Cómo tomar decisiones cuando no todo es blanco o negro.

GradosBajo / medio / altoControlReglas flexibles

Definición sencilla

La lógica difusa permite razonar con grados. En lugar de decir “frío” o “caliente” como categorías rígidas, permite valores intermedios: poco frío, templado, algo caliente o muy caliente.

Clásica vs difusa

Lógica clásica

La condición es verdadera o falsa.

Lógica difusa

Una condición puede cumplirse parcialmente.

Explicación técnica entendible

Se definen variables lingüísticas como temperatura, humedad u ocupación. Cada variable tiene conjuntos difusos como baja, media o alta. Luego se aplican reglas: “SI temperatura alta Y ocupación alta, ENTONCES ventilador fuerte”. Al final se convierte esa decisión en una salida concreta.

Conceptos clave

ConceptoSignificado
Variable lingüísticaDato expresado con palabras, como temperatura.
Conjunto difusoCategoría gradual, como frío o caliente.
Grado de pertenenciaQué tanto pertenece un valor a una categoría.
Regla difusaRelación SI/ENTONCES con grados.
DefuzzificaciónConvertir la decisión difusa en una acción numérica.

Diferencia con Machine Learning (ML)

La lógica difusa no necesita aprender miles de ejemplos. Usa reglas y rangos diseñados por personas. Conviene cuando el conocimiento se expresa como grados y se desea control interpretable.

Ejemplo desarrollado: aire acondicionado inteligente

Entrada: temperatura, humedad y ocupación del salón.

Regla: SI temperatura alta Y ocupación alta ENTONCES ventilador fuerte.

Salida: intensidad del ventilador y temperatura objetivo.

1. Leer sensores.
2. Calcular grados: frío, templado, caliente.
3. Aplicar reglas difusas.
4. Ajustar intensidad sin cambios bruscos.

Componentes

Variables, rangos, conjuntos difusos, reglas, motor de inferencia y salida ajustada.

Casos de uso

Aires acondicionados, lavadoras, frenos, control de tráfico y evaluación de riesgo gradual.

Cuándo conviene

Cuando se trabaja con grados como bajo, medio, alto; frío, templado o caliente.

Riesgos y limitaciones

Reglas mal calibradasSensores incorrectosConsumo energéticoDecisiones ambiguasRangos poco realistas

2 Sus reglas son entendibles y ajustables.
